Los pseudo-elementos de CSS 2.1 se mantienen en CSS 3, pero cambia su sintaxis y ahora se utilizan :: en vez de : delante del nombre de cada 
pseudo-elemento: ::first-line, selecciona la primera línea del texto de un elemento. ::first-letter, selecciona la primera letra del texto de un 
elemento. ::before, selecciona la parte anterior al contenido de un elemento para insertar nuevo contenido generado. ::after, selecciona la parte 
posterior al contenido de un elemento para insertar nuevo contenido generado.
::selection, selecciona el texto que ha seleccionado un usuario con su ratón o teclado
